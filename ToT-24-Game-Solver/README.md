# 24 Point Game Solver using Tree of Thoughts (ToT)

这是一个基于 **Tree of Thoughts (ToT)** 算法框架实现的 24点游戏求解器。

## 📋 任务描述
给定4个数字（1-13），使用加减乘除和括号，使结果等于24。
本项目实现了以下要求：
- 自定义思维状态（Thought State）
- BFS (广度优先) 和 DFS (深度优先) 搜索策略
- 状态评估与去重剪枝

## 🛠️ 技术栈
- **Language**: Python 3.8+
- **Algorithm**: Tree of Thoughts, BFS, DFS
